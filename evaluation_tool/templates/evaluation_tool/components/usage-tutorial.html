In drei Schritten zur individuellen Unterrichtsevaluation

So können Sie mit AMADEUS in nur drei Schritten Ihren eigenen Unterricht aus der Perspektive von Schülerinnen und Schülern evaluieren lassen:  

Schritt 1

Erstellen Sie Ihre eigene Unterrichtsevaluation für ein bestimmtes Fach oder eine bestimmte Lernendengruppe.

Jetzt können Sie ...

  • ... eine E-Mail-Adresse angeben, an welche Ihnen weitere Informationen zu Ihrer Evaluation gesendet werden.
  • ... ein Unterrichtsfach auswählen, in dem Ihr Unterricht evaluiert werden soll. Falls Sie eines der Fächer Deutsch, Englisch, Evangelische Religion, Latein, Mathematik oder Musik auswählen, werden in der Evaluation auch fachspezifische Merkmale berücksichtigt. 
    Falls Sie Ihre Evaluation in keinem der genannten Fächer durchführen möchten, wählen Sie bitte die Option allgemeine Unterrichtsevaluation. Hier werden lediglich generische Merkmale analysiert.
  • ... einen Zeitraum zu wählen, in dem Ihr Unterricht von den Lernenden evaluiert werden soll.

Schritt 2

Nachdem Sie eine Evaluation erstellt haben, wird Ihnen ein QR-Code bzw. Link angezeigt, über den Ihre Lernenden die Unterrichtsevaluation mit ihrem Smartphone oder Tablet öffnen und starten können. Teilen Sie diesen bitte mit Ihren Schülerinnen und Schülern.

Außerdem erhalten Sie eine E-Mail mit einem Link zu Ihrer persönlichen Statusseite. Auf dieser können Sie ggf. noch einmal den Share-Link bzw. Share-QR-Code aufrufen, den Status Ihrer Evaluation einsehen oder Ihre Evaluation vorzeitig beenden.

Schritt 3

Sobald Ihre Evaluation läuft, erhalten Sie auf Ihrer Statusseite eine Vorschau Ihrer Ergebnisse. Ihren individuellen Ergebnisbericht können Sie als PDF- oder Excel-Dokument herunterladen und speichern.
